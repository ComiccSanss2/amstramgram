# Amstramgram 

[cite_start]Projet de réseau social (type Instagram)[cite: 1, 4]. 
[cite_start]L'application repose sur une architecture moderne utilisant obligatoirement **TypeScript** pour le Front et le Back.

##  Stack Technique
* [cite_start]**Frontend** : React + TypeScript[cite: 6].
* [cite_start]**Backend** : Node.js + TypeScript[cite: 7].
* [cite_start]**Gestion de version** : Git.

---

## Installation et Lancement

### 1. Prérequis
* Node.js (v18 ou supérieur)
* npm ou yarn

### 2. Installation
Ouvrez deux terminaux différents :

**Pour le Frontend :**
```bash
cd client
npm install
```


**Pour le Backend :**
```bash
cd server
npm install
```
